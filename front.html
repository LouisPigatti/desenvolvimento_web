Front-End

HTML Tags

  • Html>: Elemento raiz que envolve todo o conteúdo da página.
  • head>: Contém informações sobre a página, como título, metadados e links para estilos.
  • meta>: Define metadados, como o conjunto de caracteres utilizado.
  • title>: Define o título da página exibido na aba do navegador.
  • body>: Contém o conteúdo visível da página.
  • h1>, p>, a>, img>, ul>, li>, div>: Tags para estruturar e exibir texto, links, imagens e listas.

CSS Conceitos

  • Seletores: São usados para escolher elementos HTML a serem estilizados.
  • Propriedades: Definem os estilos a serem aplicados, como color (cor do texto) ou font-size (tamanho da fonte).
  • Valores: São atribuídos às propriedades para definir como os estilos devem ser aplicados.

JavaScript Conceitos

  • Variáveis: São usadas para armazenar dados.
  • Funções: São blocos de código reutilizáveis que executam tarefas específicas.
  • Eventos: Permitem que o código responda a ações do usuário, como cliques ou envio de formulários.
  • Manipulação do DOM: Permite interagir com elementos HTML e alterar seu conteúdo, estilo e estrutura.
