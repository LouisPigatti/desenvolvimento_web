Protocolo HTTP

O HTTP é um protocolo de comunicação usado para transferir dados na World Wide Web. Ele funciona no modelo cliente-servidor, onde um cliente (geralmente um navegador web) faz solicitações a um servidor web para obter recursos, como páginas da web.

Métodos HTTP Comuns

  • GET: Usado para solicitar dados de um recurso específico no servidor. Por exemplo, ao abrir uma página da web, o navegador faz uma solicitação GET para obter o conteúdo da página.
  • POST: Usado para enviar dados ao servidor, geralmente para criar ou atualizar recursos. Por exemplo, quando você preenche um formulário em um site e o envia, uma solicitação POST é usada para enviar os dados do formulário ao servidor.
  • PUT: Usado para atualizar um recurso específico no servidor. É semelhante ao POST, mas geralmente usado para atualizações completas de recursos.
  • DELETE: Usado para excluir um recurso específico no servidor. Quando uma solicitação DELETE é feita, o servidor remove o recurso correspondente.

Solicitação HTTP

  • Método: Indica qual ação deve ser realizada no recurso.
  • URI (Uniform Resource Identifier): O endereço do recurso solicitado.
  • Cabeçalhos: Informações adicionais, como tipo de conteúdo ou informações de autenticação.
  • Corpo da Solicitação: Opcional e geralmente contém dados enviados ao servidor, como em uma solicitação POST.

Resposta HTTP

  • Status: Indica o resultado da solicitação.
  • Cabeçalhos: Informações adicionais, como tipo de conteúdo e tamanho do conteúdo.
  • Corpo da Resposta: O conteúdo retornado pelo servidor.
